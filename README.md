# CGPA Calculator (C++)

This is a simple C++ program that calculates the CGPA (Cumulative Grade Point Average) based on user input.

## 📌 Features
- Takes input for the number of courses
- Accepts grade points and credit hours for each course
- Calculates:
  - Total Credits
  - Total Grade Points
  - Final CGPA
- Displays results clearly to the user

## 🧮 How It Works
1. User enters the number of courses.
2. For each course:
   - Enter the grade (e.g., 4.0, 3.7, etc.)
   - Enter the credit hours
3. The program calculates:
   - **Grade Points = Grade × Credit Hours**
   - **CGPA = Total Grade Points / Total Credits**
4. Outputs the final CGPA.
